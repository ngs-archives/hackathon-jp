本ドメインは、2009/3/20 に Google Tokyo office で行われた Android Hackathon で、
Nikon UP と Android を組み合わせて、ドラゴンボールのスカウター（＋ドラゴンレーダー）
を作った際の、戦いの跡地です。

本サイトは未完成です。完成させようと思ったのですが、私の未熟さが原因で、
つっこみどころが多い作りであることがわかり、別ドメインで汎用的なサイトを
作り直すことにしました。

ドメインは、http://gps-open-logger.appspot.com/ です。

今後、AndroidやiPhoneでGPSアプリを作りたい人、よかったら上記ＵＲＬをご自由にお使いいただければ幸いです。

以下は、Hackathonの時に作った本サイトの仕様です。




■android-scouter API 本番向けやっつけ仕様（暫定）

○叩き方

http://android-scouter.appspot.com/data/?id=XXXXXX&geo_x=XX.XXXX&geo_y=XX.XXXX&range=1000

id     number device id 
geo_x  float  経度
geo_y  float  緯度　
range  number 知りたい範囲（m）

○戻り

result              string  OK, NG
timestamp           string  レスポンスを発行した日時
message             string  エラーメッセージ（resultがNGの場合のみ）
data                xml     レスポンスデータ
data/id             number  device id
data/name           string  ハンドル名
data/geo_x          float   経度
data/geo_y          float   緯度
data/picture        string  アイコン画像URL
data/power          number  パワー
data/profile        string  プロフィール
data/modified       string  最終更新日
data/created        string  登録日


例.NGの場合

<?xml version="1.0" encoding="UTF-8"?>
<response>
    <result>NG</result>
    <timestamp>2009-03-03 15:30:33</timestamp>
    <message>このidは登録されていません</message>
</response>
